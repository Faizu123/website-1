SproutCore Dev Tools on Windows with One-Click

Download the Installer

The SproutCore Installer for Windows currently requires Windows 7 or Vista. XP support is coming soon. Download Now!

Using the Dev Tools

  1. The SproutCore Dev Tools run via a special Windows Command Prompt. To start the prompt, run SproutCore/SproutCore Prompt from the Start Menu.
  2. From the command prompt you can use any of the sc-* tools (sc-init, sc-gen, sc-server, ...)

Advanced Installation

If you already do Ruby development on Windows or would just like to have greater flexibility, you can install the SproutCore Dev Tools as a Ruby gem. If you want to install Ruby, start on Step A. If you already have a Ruby installation, skip to Step B.

Step A: Installing Ruby

There are multiple options for installing Ruby on Windows. We recommend using either the One-Click Ruby Installer or Cygwin with RVM.

One-Click Ruby Installer

  1. First download and install Ruby 1.9.2 from http://rubyinstaller.org/downloads/.
  2. Next open "Ruby-1.9.2-*/Command Prompt with Ruby" from your Start Menu. (If you installed to your system $PATH, you can just open a normal command prompt.)
  3. Continue on to Step B to install SproutCore Dev Tools.

Cygwin with RVM

This option gives you the most flexibility but is also the most work to set up.

Install Cygwin

  1. Begin by downloading the Cygwin setup.exe from http://www.cygwin.com/
  2. When selecting packages for Cygwin make sure to select the following:
    • Devel->gcc-core
    • Devel->gmake
    • Devel->gzlib-devel
    • Net->gcurl
    • Net->gopenss
    • Perl->gperl
    • Utils->gncurses
  3. Next you'll need to set up proper SSL certs. To do so, run the following code: 
    $ mkdir -p /usr/ssl/certs
      $ cd /usr/ssl/certs
      $ curl http://curl.haxx.se/ca/cacert.pem | awk 'split_after==1{n++;split_after=0} /-----END CERTIFICATE-----/ {split_after=1} {print > "cert" n ".pem"}'
      $ c_rehash

Install RVM and Ruby

  1. With Git: 
    $ bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)
  2. Without Git: 
    $ curl -s https://rvm.beginrescueend.com/install/rvm -o rvm-installer
    $ chmod +x rvm-installer
    $ ./rvm-installer --version latest
  3. Install Ruby: 
    $ rvm install ruby-1.9.2
    $ rvm use 1.9.2
  4. If you would like to set this as your primary Ruby version, run: 
    $ rvm --default use 1.9.2
  5. Continue to Step B

Step B: Installing SproutCore

This is the easy part, just run:

$ gem install sproutcore

From the Cygwin prompt you can now use any of the sc-* tools (sc-init, sc-gen, sc-server, ...)
