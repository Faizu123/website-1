Building Your First SproutCore App Is Easier Than You Think. Here's How...

Installing SproutCore

The easiest way to install SproutCore is as a RubyGem:

$ [sudo] gem install sproutcore

Creating Your First Application

  1. $ sc-init HelloWorld
    ~ Created directory at hello_world
    ~ Created file at hello_world/Buildfile
    ~ Created file at hello_world/README
    ~ Created directory at apps
    ~ Created directory at apps/hello_world
    ~ Created file at apps/hello_world/core.js
    ~ Created file at apps/hello_world/main.js
    ~ Created directory at apps/hello_world/resources
    ~ Created file at apps/hello_world/resources/loading.rhtml
    ~ Created file at apps/hello_world/resources/main_page.js
    Your new SproutCore project is ready!

    To get started, you can find your initial application in the "apps" directory.
  2. $ cd hello_world
  3. $ sc-server
    SproutCore v1.0.1037 Development Server
    Starting server at http://0.0.0.0:4020 in debug mode
    To quit sc-server, press Control-C
    >> Thin web server (v1.2.5 codename This Is Not A Web Server)
    >> Maximum connections set to 1024
    >> Listening on 0.0.0.0:4020, CTRL+C to stop
  4. Open http://localhost:4020/hello_world in your browser and you should see the following:
  5. Congratulations, you've just built your first SproutCore application!

Installing the SproutCore Dev Tools has never been easier!

The SproutCore Dev Tools are supported by all major platforms.

While a compiled SproutCore app is pure JavaScript, HTML and CSS and can run in any modern browser, you'll still want to install our Dev Tools to make things easier while creating your app. The SproutCore Dev Tools run on all major platforms, including Windows, Mac OS X, and Linux.

SproutCore on Windows

SproutCore can be installed on Windows via an easy one-click installer or, for more advanced users, as a Ruby gem. Install Now!

SproutCore on Mac OS X

SproutCore can be installed on Mac OS X via an easy package installer or, for more advanced users, as a Ruby gem. Install Now!

SproutCore on Linux

SproutCore can be installed on most Linux systems as a Ruby gem. Install Now!

SproutCore on Other Platforms

While other platforms are not officially supported, you may be able to install SproutCore as a Ruby gem. Learn More!
