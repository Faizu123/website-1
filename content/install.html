Installing SproutCore has never been easier!

SproutCore is supported by all major platforms.

While a compiled SproutCore app is pure JavaScript, HTML and CSS and can run in any modern browser, you'll still want to install our Build Tools to make things easier while creating your app. SproutCore runs on all major platforms, including Windows, Mac OS X, and Linux.
