Installation

If you already do Ruby development on Windows or would just like to have greater flexibility, you can install the SproutCore Dev Tools as a Ruby gem. If you want to install Ruby, start on Part A. If you already have a Ruby 1.9.3 installation, skip to Part B.

Part A: Installing Ruby

There are multiple options for installing Ruby on Windows. We recommend using either the One-Click Ruby Installer or Cygwin with RVM.

Easiest: One-Click Ruby Installer

  1. First download and install Ruby 1.9.3 from http://rubyinstaller.org/downloads/.
  2. Next open "Ruby-1.9.3-*/Command Prompt with Ruby" from your Start Menu. (If you installed to your system $PATH, you can just open a normal command prompt.)
  3. Continue to Part B.

Most Flexibility: Cygwin with RVM

This option gives you the most flexibility but is also the most work to set up.

Install Cygwin

  1. Begin by downloading the Cygwin setup.exe from http://www.cygwin.com/
  2. When selecting packages for Cygwin make sure to select the following:
    • Devel->gcc-core
    • Devel->gmake
    • Devel->gzlib-devel
    • Net->gcurl
    • Net->gopenss
    • Perl->gperl
    • Utils->gncurses
  3. Next you'll need to set up proper SSL certs. To do so, run the following code: 
    $ mkdir -p /usr/ssl/certs
$ cd /usr/ssl/certs
$ curl http://curl.haxx.se/ca/cacert.pem | awk 'split_after==1{n++;split_after=0} /-----END CERTIFICATE-----/ {split_after=1} {print > "cert" n ".pem"}'
$ c_rehash

Install RVM and Ruby

  1. Install RVM along with Ruby 1.9.3. 
    $ \curl -#L https://get.rvm.io | bash -s stable --autolibs=3 --ruby=1.9.3

Part B: Installing SproutCore

  1. This part is easy. 
    $ gem install sproutcore
  2. Proceed to the tutorial.
