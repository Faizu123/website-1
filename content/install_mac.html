Installation

If you already do Ruby development on your Mac or would just like to have greater flexibility, you can install the SproutCore Dev Tools as a Ruby gem.

SproutCore's gem requires Ruby 1.9.3 or later. You can test your system's installed version with ruby -v. If you have a version prior to 1.9.3, follow these instructions to upgrade using rvm (Ruby Version Manager). If you are already running ruby 1.9.3 or later, skip to Step 3.

Notes for installing command line developer tools

The command line developer tools required for installation are no longer included by default with Xcode.

Note for OS 10.8 - Mountain Lion
Download separately from within Xcode or directly from Apple's Developer Site.

Note for OS 10.9 - Mavericks
Install by running xcode-select --install in a terminal or download directly from Apple's Developer Site.

Installing Ruby and SproutCore

  1. Install RVM along with Ruby 1.9.3. 
    $ \curl -L https://get.rvm.io | bash -s stable --autolibs=3 --ruby=1.9.3
  2. Add the following to your ~/.bash_profile so the build tools will be in your path. 
    [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" # This loads RVM into a shell session
  3. Install SproutCore: This part is easy. 
    $ gem install sproutcore
  4. Proceed to the tutorial.
