Installation via Ruby Gem

To get started with SproutCore on Linux, you'll need a Ruby installation. If you already have a Ruby installation, skip to Step B.

Step A: Installing RVM and Ruby

  1. If you have Git: 
    $ bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)

    Without Git: 
    $ curl -s https://rvm.beginrescueend.com/install/rvm -o rvm-installer
$ chmod +x rvm-installer
$ ./rvm-installer --version latest
  2. Add the following to your ~/.bash_profile to add Ruby to your PATH. 
    [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" # This loads RVM into a shell session
  3. Install Ruby: 
    $ rvm install ruby-1.9.2
$ rvm use 1.9.2
$rvm --default use 1.9.2
  4. Install SproutCore: This is the easy part. 
    $ gem install sproutcore
  5. Proceed to the tutorial
