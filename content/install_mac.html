SproutCore Dev Tools on Mac OS X with Package

Download the Package

The SproutCore Package for Mac OS X runs on all recent versions of OS X. Download Now!

Using the Dev Tools

The SproutCore Dev Tools run via the Terminal. From the terminal you can use any of the sc-* tools (sc-init, sc-gen, sc-server, ...)

Advanced Installation

If you already do Ruby development on your Mac or would just like to have greater flexibility, you can install the SproutCore Dev Tools as a Ruby gem. However, it is not recommended that you use the system installation, so we show you how to install with RVM. If you already have a non-system Ruby installation, skip to Step B.

Step A: Installing RVM and Ruby

  1. With Git: 
    $ bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)
  2. Without Git: 
    $ curl -s https://rvm.beginrescueend.com/install/rvm -o rvm-installer
$ chmod +x rvm-installer
$ ./rvm-installer --version latest
  3. If installing for a single user instead of root, add the following to your ~/.bash_profile 
    [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" # This loads RVM into a shell session
  4. Install Ruby: 
    $ rvm install ruby-1.9.2
$ rvm use 1.9.2
  5. If you would like to set this as your primary Ruby version, run: 
    $ rvm --default use 1.9.2
  6. Continue to Step B

Step B: Installing SproutCore

This is the easy part, just run:

$ gem install sproutcore


From the Terminal prompt you can now use any of the sc-* tools (sc-init, sc-gen, sc-server, ...)
